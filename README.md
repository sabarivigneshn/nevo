# nevo
 on-demand scheduling of EV test drives to customers.

# Frontend
Follow the below steps to start the frontend app

cd client
npm install
npm start

Application will be started in http://localhost:3000

# Backend

Follow the below steps to start the frontend app

cd server
npm install
npm start

Application will be started in http://localhost:9000

# API 

Baseurl - http://localhost:9000/api

Sample Curl Request

curl --location 'http://localhost:9000/api/vehicles/'

For more information about the APIs, use the postman collection attached in the repo.
